Conectar WhatsApp

Para conectar via QR Code, acesse o Control UI do OpenClaw e escaneie o código com o WhatsApp do número dedicado.

1
Abra o Control UI do OpenClaw
2
Vá em Canais → WhatsApp
3
Escaneie o QR Code com o celular do número da academia
Abrir Control UI →

Execute o script de pareamento no servidor e cole o código de 8 dígitos abaixo:

node scripts/whatsapp-pairing-code.mjs <phone> <account_id>

Digite este código no WhatsApp:

WhatsApp → Configurações → Aparelhos Conectados → Conectar com número de telefone
